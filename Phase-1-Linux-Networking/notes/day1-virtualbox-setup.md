# ==========================================
# DAY 1 : VIRTUALBOX INSTALLATION
# ==========================================

# OBJECTIVE:
# VirtualBox install karna aur virtualization ON hai ya nahi check karna.

# ------------------------------------------
# STEP 1 : Virtualization Check Karna
# ------------------------------------------

# Windows me:
# Open Task Manager

Ctrl + Shift + Esc

# Fir:
# Performance Tab > CPU

# Waha check karo:
# "Virtualization : Enabled"

# Agar Enabled hai:
# ✅ System ready hai

# Agar Disabled hai:
# ❌ BIOS/UEFI me jaakar enable karna padega


# ------------------------------------------
# STEP 2 : VirtualBox Download Karna
# ------------------------------------------

# Official Website:
https://www.virtualbox.org/

# Download:
# "Windows Hosts" version


# ------------------------------------------
# STEP 3 : VirtualBox Install Karna
# ------------------------------------------

# Downloaded .exe file par double click karo

# Installation Steps:
# Next > Next > Next > Install

# Agar warning aaye:
# "Network Interface Warning"
# to YES press karo

# Installation complete hone ke baad:
# Finish


# ------------------------------------------
# STEP 4 : VirtualBox Open Karna
# ------------------------------------------

# Start Menu me search karo:

VirtualBox

# Open karo

# Agar successfully open ho gaya:
# ✅ Installation successful


# ------------------------------------------
# IMPORTANT NOTES
# ------------------------------------------

# Virtualization OFF hone par:
# Virtual machines properly run nahi hongi

# VirtualBox use hota hai:
# Multiple Operating Systems run karne ke liye
# Example:
# Kali Linux
# Ubuntu
# Windows

# Cyber Security me:
# Safe Testing Environment banane ke liye use hota hai
